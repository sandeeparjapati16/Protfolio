# Sandeep Web Studio Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript to showcase skills, education, projects, and contact details.

## Overview

This project is a responsive portfolio website for Sandeep Kumar. It includes a sidebar with contact information, a multi-section main content area, interactive navigation, project filtering, and a contact form that opens WhatsApp for messaging.

## Features

- Clean, modern portfolio layout
- Responsive sidebar navigation for desktop and mobile
- About section with personal summary and services
- Resume section with education, experience, and skill progress bars
- Portfolio section with project filtering by category
- Projects section listing academic and personal projects
- Contact section with embedded Google Map and WhatsApp contact form
- Downloadable CV PDF

## Technologies Used

- HTML5
- CSS3
- JavaScript (vanilla)
- Ionicons icon library
- Google Fonts (Poppins)

## Project Structure

- `index.html` - main page structure and content
- `assets/css/style.css` - styling for layout, responsive behavior, and visual design
- `assets/js/script.js` - interactive behavior, sidebar toggle, page navigation, filtering, and form handling
- `assets/images/` - project images, icons, avatar, and `Sandeep CV.pdf`

## How to Use

1. Open `index.html` in a web browser.
2. Use the top navigation buttons to switch sections: About, Resume, Portfolio, Projects, Contact.
3. In the Portfolio section, filter projects by category using the buttons or select dropdown.
4. Use the contact form in the Contact section to send a message via WhatsApp.
5. Download the CV from the About section using the `Download CV` button.

## Notes

- The contact form is configured to open WhatsApp with a pre-filled message.
- The project does not require a build process or server; it runs directly in the browser.
- Ionicons assets are loaded from a CDN.

## Customization

To customize the site:

- Edit `index.html` to update text, links, and project entries.
- Update `assets/css/style.css` for visual changes and layout adjustments.
- Modify `assets/js/script.js` to change behavior such as filtering and navigation.
